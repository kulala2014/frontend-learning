class

        
class是在ES6引入的语法糖，实际的底层还是会转化为原型链对象继承

                
            class Person {
                constructor(first, last, age, gender, interests) {
                    this.name = {
                        first,
                        last
                    };
                    this.age = age;
                    this.gender = gender;
                    this.interests = interests;
                }

                greeting() {
                    console.log(`HI!~ I'm ${this.name.first}`);
                };
                farewell() {
                    console.log(`${this.name.first} has left the building,byb!`);
                };
            }

            class Teacher extends Person {
                constructor(first, last, age, gender, interests, subject, grade) {
                    super(first, last, age, gender, interests);
                    this.subject = subject;
                    this.grade = grade;
                }
            }
    
        
Getters and Setters

        
有时我们可能想要更改我们创建的类中属性的值，或者我们不知道属性的最终值是什么。
            我们可以使用 getter 和 setter 来处理这种情况。
            getter 和 setter 成对工作。 getter 返回变量的当前值，其相应的 setter 将变量的值更改为它定义的值。

            
                class Teacher extends Person {
                    constructor(first, last, age, gender, interests, subject, grade) {
                        super(first, last, age, gender, interests);
                        this._subject = subject;
                        this.grade = grade;
                    }
        
                    get subject() {
                        return this._subject;
                    };
                    set subject(newSubject) {
                        this._subject = newSubject;
                    };
        
                }
